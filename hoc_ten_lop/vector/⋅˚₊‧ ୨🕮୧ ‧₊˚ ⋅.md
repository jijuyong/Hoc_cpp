# 13 - Giới thiệu về STL (Standard Template Library)

**STL (Standard Template Library)** là một phần rất quan trọng của thư viện chuẩn trong C++. Đây là tập hợp các **lớp (Classes)** và **hàm (Functions)** được xây dựng sẵn, giúp lập trình viên không cần tự cài đặt lại các cấu trúc dữ liệu hay thuật toán phổ biến.

Sử dụng STL giúp:

- 🚀 Viết chương trình nhanh hơn.
- 🛡️ An toàn hơn.
- 📖 Dễ đọc và dễ bảo trì.
- ⚡ Hiệu quả vì các thuật toán đã được tối ưu.

STL gồm ba thành phần chính:

1. **Containers (Các cấu trúc lưu trữ dữ liệu)**
2. **Iterators (Bộ lặp)**
3. **Algorithms (Các thuật toán)**

---

# 1. Containers (Các cấu trúc dữ liệu)

Container là nơi dùng để lưu trữ nhiều phần tử.

Mỗi container phù hợp với một mục đích khác nhau.

## a. std::vector

`vector` là **mảng động**.

Khác với mảng thông thường:

```cpp
int arr[10];
```

thì `vector` có thể tự động tăng hoặc giảm kích thước.

Ví dụ

```cpp
#include <vector>

std::vector<int> numbers;

numbers.push_back(10);
numbers.push_back(20);
numbers.push_back(30);
```

Kết quả

```
10 20 30
```

Truy cập giống mảng

```cpp
numbers[0]
numbers[1]
numbers[2]
```

### Ưu điểm

- Truy cập nhanh bằng chỉ số `[]`.
- Thêm phần tử cuối rất nhanh (`push_back()`).
- Không cần biết trước kích thước.

---

## b. std::deque

Deque (**Double Ended Queue**) là hàng đợi hai đầu.

Có thể thêm hoặc xóa ở:

- đầu
- cuối

đều rất nhanh.

Ví dụ

```cpp
std::deque<int> dq;

dq.push_front(1);

dq.push_back(2);

dq.push_back(3);
```

Kết quả

```
1 2 3
```

---

## c. std::list

`list` là **Danh sách liên kết kép (Doubly Linked List).**

Ví dụ

```cpp
std::list<int> lst;

lst.push_back(10);
lst.push_back(20);
lst.push_back(30);
```

### Ưu điểm

- Thêm hoặc xóa giữa danh sách rất nhanh.

### Nhược điểm

Không hỗ trợ

```cpp
lst[2];
```

Muốn truy cập phải đi từng phần tử.

---

# Container kết hợp (Associative Containers)

Những container này lưu dữ liệu theo **khóa (Key).**

---

## d. std::set

`set` lưu:

- Không có phần tử trùng nhau.
- Tự động sắp xếp tăng dần.

Ví dụ

```cpp
std::set<int> s;

s.insert(5);
s.insert(3);
s.insert(5);
s.insert(2);
```

Kết quả

```
2 3 5
```

Số 5 chỉ xuất hiện một lần.

---

## e. std::map

`map` giống như một **từ điển**.

Mỗi dữ liệu gồm:

```
Key -> Value
```

Ví dụ

```cpp
std::map<std::string,int> score;

score["Math"] = 90;

score["English"] = 80;
```

Kết quả

```
Math -> 90

English -> 80
```

Tìm dữ liệu rất nhanh theo Key.

---

# Unordered Containers

Các container này sử dụng **Hash Table**.

Khác với `set` và `map`:

- Không sắp xếp dữ liệu.
- Tốc độ tìm kiếm trung bình nhanh hơn.

---

## f. unordered_set

Giống `set`

Nhưng:

```
Không sắp xếp
```

Ví dụ

```cpp
std::unordered_set<int> us;
```

---

## g. unordered_map

Giống `map`

Nhưng:

```
Không sắp xếp Key
```

Ví dụ

```cpp
std::unordered_map<std::string,int> age;
```

---

# 2. Iterator (Bộ lặp)

Iterator giúp chúng ta duyệt các phần tử trong Container.

Có thể xem Iterator giống như một **con trỏ (Pointer).**

Nó có thể:

- trỏ tới phần tử
- di chuyển sang phần tử tiếp theo

---

## begin()

Trả về Iterator tới phần tử đầu tiên.

```cpp
v.begin()
```

---

## end()

Trả về Iterator tới vị trí **ngay sau phần tử cuối cùng.**

Lưu ý:

```
end()

KHÔNG phải là phần tử cuối.
```

Nó chỉ là điểm dừng của vòng lặp.

---

Ví dụ

```cpp
std::vector<int> v={10,20,30};

for(std::vector<int>::iterator it=v.begin();
    it!=v.end();
    ++it)
{
    std::cout<<*it<<" ";
}
```

Kết quả

```
10 20 30
```

Ở đây

```cpp
*it
```

lấy giá trị của Iterator.

```cpp
++it
```

di chuyển sang phần tử tiếp theo.

---

# Range-based for

Trong C++11 có cách viết ngắn hơn.

```cpp
for(int x : v)
{
    std::cout<<x<<" ";
}
```

Kết quả

```
10 20 30
```

Bên trong, C++ vẫn sử dụng Iterator.

Đây là cách viết hiện đại và dễ đọc hơn.

---

# 3. Algorithms (Thuật toán)

Thư viện

```cpp
#include <algorithm>
```

cung cấp rất nhiều thuật toán có sẵn.

Chúng hoạt động với Iterator.

---

## std::sort()

Sắp xếp dữ liệu.

```cpp
std::sort(v.begin(),v.end());
```

Ví dụ

```cpp
std::vector<int> v={5,2,8,1};

std::sort(v.begin(),v.end());
```

Kết quả

```
1 2 5 8
```

---

## std::find()

Tìm phần tử.

```cpp
auto it=std::find(v.begin(),v.end(),8);
```

Nếu tìm thấy

```
it != v.end()
```

---

## std::reverse()

Đảo ngược dữ liệu.

```cpp
std::reverse(v.begin(),v.end());
```

Ví dụ

```
1 2 3 4
```

↓

```
4 3 2 1
```

---

## std::count()

Đếm số lần xuất hiện.

```cpp
int c = std::count(v.begin(),v.end(),5);
```

Ví dụ

```
5 3 5 2 5
```

Kết quả

```
3
```

---

Ngoài ra STL còn rất nhiều thuật toán khác:

- `copy()`
- `remove()`
- `transform()`
- `max_element()`
- `min_element()`
- `binary_search()`
- ...

Thay vì tự viết vòng lặp, nên ưu tiên dùng STL vì:

- nhanh hơn
- ít lỗi hơn
- tối ưu hơn

---

# Ví dụ hoàn chỉnh

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
    vector<int> numbers={5,1,9,2,7};

    sort(numbers.begin(),numbers.end());

    for(int x : numbers)
    {
        cout<<x<<" ";
    }

    return 0;
}
```

Kết quả

```
1 2 5 7 9
```

# Tổng kết

| Thành phần | Chức năng |
|------------|-----------|
| `vector` | Mảng động |
| `deque` | Hàng đợi hai đầu |
| `list` | Danh sách liên kết |
| `set` | Lưu giá trị duy nhất, tự sắp xếp |
| `map` | Lưu cặp Key - Value |
| `unordered_set` | Set không sắp xếp |
| `unordered_map` | Map không sắp xếp |
| `iterator` | Duyệt qua các phần tử |
| `algorithm` | Thuật toán có sẵn |

---

# Những điều cần nhớ

- 📦 **Container** dùng để lưu dữ liệu.
- 👉 **Iterator** giúp duyệt các phần tử.
- ⚙️ **Algorithm** giúp xử lý dữ liệu.
- 🚀 `vector` là container được sử dụng nhiều nhất.
- 📚 STL giúp viết chương trình ngắn gọn, nhanh và an toàn hơn.

# Bài tập

## Bài 1. Làm việc với Vector

1. Tạo:

```cpp
std::vector<int>
```

2. Thêm vào 10 số ngẫu nhiên.

3. Dùng

```cpp
std::sort()
```

để sắp xếp.

4. In toàn bộ Vector bằng vòng lặp:

```cpp
for(int x : vector)
```

---

## Bài 2. Danh sách mua sắm

Tạo:

```cpp
std::map<std::string,int>
```

để lưu:

```
Tên sản phẩm -> Số lượng
```

Ví dụ

```
Bread -> 2

Milk -> 1

Egg -> 12
```

Yêu cầu:

- Thêm vài sản phẩm.
- Kiểm tra xem có `"Bread"` hay không.
- In toàn bộ danh sách.

---

## Bài 3. Loại bỏ số trùng

Cho mảng:

```cpp
{1,2,2,3,1,4}
```

Đưa tất cả các phần tử vào:

```cpp
std::set<int>
```

Sau đó in kết quả.

Kết quả mong muốn:

```
1 2 3 4
```

Quan sát rằng `set` tự động loại bỏ các phần tử trùng lặp và sắp xếp theo thứ tự tăng dần.
